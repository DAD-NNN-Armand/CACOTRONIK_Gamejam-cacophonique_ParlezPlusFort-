# ParlezPlusFort!!!!
Jeu vidéo expérimental réalisé dans le cadre du workshop "Cacotronik : Gamejam cacophonique", supervisé par Chloé Desmoineaux et Robin Moretti.

## Équipe :
- **Armand Popot** Enregistrements sonores
- **Dana Durarenko** Programmation
- **Thalie Creuset--Emery** Ingénierie son
- **Zhuoer Chen** Scenographie

## Mode d’emploi :
Les joueur·euses sont invité·es à s’installer au pupitre, à répondre dans le microphone aux questions des journalistes transmises par l’enceinte et à appuyer sur le bouton afin de relancer la partie. 

## Scenario :
C’est la salle de _Prise de Parole Flash_ (_P.P.F_). Le joueur·euses incarnent un politicien assailli par le flash et les questions insistantes d’une dizaine de journalistes désagréables, qui lui demandent sans cesse d'ajuster son volume.
